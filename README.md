# Schema-Driven Form Component (v1)

This repository contains a schema-driven form component built with Next.js, TypeScript, React Hook Form and Zod. It renders create/edit forms from JSON Schema (2020-12) with UI extensions.

Quick start

1. Install dependencies

```pwsh
npm install
```

2. Run dev server

```pwsh
npm run dev
```

3. Run unit tests

```pwsh
npm run test
```

Docs: See `specs/001-schema-driven-form/` for the feature spec, plan, data model, and tasks.
